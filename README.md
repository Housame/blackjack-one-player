# Blackjack – 1 spelare
Ett enkelt, stilrent Blackjack byggt med **HTML, CSS och JavaScript**. Kortbilderna hämtas via **länkar** från `deckofcardsapi.com` så inga bildfiler behövs.

## Så kör du
1. Öppna `index.html` i din webbläsare.
2. Sätt insats och klicka **Dela**.
3. Använd **Hit** för att ta kort och **Stand** för att stanna. Dealern drar till 17.

## Struktur
- `index.html` – markup och layout
- `style.css` – fin, modern design
- `script.js` – spelregler och logik
